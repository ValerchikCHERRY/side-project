Здесь видео с работой приложения. Есть рабочая база данных на PostgreSQL17, баланс, пополняемый через платежного бота в телеграм (платежи принимаются в TONCOIN на счет в Tonkeeper), транзакции отслеживаются через специальный API от Telegram. Пока только не готов блок с отрытием сделок на реальной бирже. MVP проекта готово на 65%. Ввиду перебоев в работе Telegram, приложение работает очень медленно (оно хостится на моем ноутбуке).
